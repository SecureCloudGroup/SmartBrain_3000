# SmartBrain_3000 documentation

These pages are the user guide. They're the **single source** for both the public
docs (here on GitHub) and the in-app **Help** page (`/help`) — the app renders
this same Markdown at build time.

1. [Getting started](01-getting-started.md) — requirements, install, first run, locking, updating.
2. [Connect a model](02-models.md) — cloud provider keys and local models (Ollama / MLX).
3. [Using SmartBrain_3000](03-features.md) — chat + tools, knowledge, planner, schedules, email, usage &amp; cost, activity.
4. [Share knowledge with Vaults](04-vaults.md) — group documents, share a sealed vault privately, publish one publicly, subscribe by URL with verified updates.
5. [Connect external tools (MCP)](05-mcp.md) — expose read-only Knowledge to a desktop AI client (e.g. Claude Desktop, Cursor).
6. [Backup &amp; recovery](06-backup-recovery.md) — export, encrypted backup, restore, change passphrase, Recovery Key.
7. [Privacy &amp; security](07-privacy-security.md) — what protects your data, what leaves your machine, real world limits.
8. [Remote access](08-remote-access.md) — reach your assistant from your phone, on Wi-Fi or cellular, by pairing a device (WebRTC, no port-forward, off by default).
9. [Design limits](09-design-limits.md) — the intentional single-user / local-first scope choices and why we made them.
